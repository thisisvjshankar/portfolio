Case Study 01 · Citigroup
Impact Analysis Application
Infosys via Citigroup · 2022–Present
🔍 Problem
Citi had 150+ downstream systems dependent on upstream data feeds. Any upstream change could silently break downstream applications — with no visibility into what would be affected.
⚙️ Approach
Led full requirements gathering and product management. Mapped all upstream-downstream dependencies, facilitated design workshops with multiple SMEs, authored BRD and user stories.
✅ Outcome
150+ downstream teams now have full impact visibility before releases — enabling faster, safer deployments across Citi Tech.
Case Study 02 · HSBC Bank
Oracle Fusion — Test Automation
Infosys via HSBC · 2018–2019
🔍 Problem
Every quarterly upgrade required a full manual regression cycle — consuming enormous time with inconsistent coverage across Procurement, Expenses, and Fixed Assets modules.
⚙️ Approach
Led automation initiative using Tricentis TOSCA. Designed a reusable regression pack, defined test scope, coordinated the team, and built the business case for senior stakeholders.
✅ Outcome
Saved 3 full-time testing efforts per cycle. Reduced manual testing by 45% — reusable suite now runs across all future upgrades.
